# HUGS Website (Netlify / GitHub)

Static site for the HUGS initiative.

## Deploy with Netlify
1. Push this repo to GitHub
2. Netlify → Add new site → Import from GitHub
3. Select this repo
4. Build command: NONE
5. Publish directory: /

## Edit
- Replace `assets/hugs-logo.jpg` if logo updates
- Edit text directly in `index.html`
